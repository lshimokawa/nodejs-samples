# Hello World en Node.js
Ejemplo en consola, HTTP Server y TCP Server.

## Consola
    node hello-world-console.js

## HTTP Server
    node hello-world-http-server.js

Abrir un browser en:
    http://localhost:8000/

## TCP Server
    node hello-world-tcp-server.js 

Abrir una consola:
    $ telnet localhost 8000
